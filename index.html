Que es un Hosting?

Un hosting es un servicio de alojamiento de sitios web en servidores físicos remotos, allí se almacenan todos los archivos y el contenido de tu página para que este disponible para todo el mundo las 24 horas del día. Es comparable a alquilar un local o galpón para poner tu comercio. Mas acerca de los Hostings.

Que es un dominio?

Un dominio web es una dirección única e irrepetible de un sitio web, por ej www.mzdev.com es el dominio de esta página. Es comparable a la dirección de una vivienda o una patente de auto. Mas acerca de los Dominios.

Como puedo administrar mi sitio yo mismo?

Por el momento solo ofrezco sitios web estáticos, los cuales solo pueden ser modificados si se tiene el conocimiento necesario en desarrollo web.
